Versão Utilizada do Python: 3.9

Para funcionar o código, você precisar instalar os seguintes pacotes:

DateTime
sqlite3
requests


O código foi testado no Pycharm. Caso você precise e utilize no terminal, use os comandos:

pip3 install datetime.

pip3 install sqlite3.

pip3 install requests.

Para ver o banco de dados, baixe o DB Browser for SQLite. Recomendo que use o Pycharm para rodar o código. Pois o mesmo já vem a maioria dos recursos necessários.
